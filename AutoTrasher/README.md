# AutoTrasher

自動回收放在裡面的指定物品

[Spigot](https://www.spigotmc.org/resources/auto-trasher-keep-your-inventory-clean-1-8-1-16.46037/)

## Triton設定 Config

**Prefix:** `AutoTrasher.config.`  
**Variable Regex:** `%\w+%`  
**Ignored keys:**

```
UpdateCheckerEnabled
glow
material
craftable
shaped
recipe.+
RequireInInventory
ignoreDurability
size
lockedSlot\.material
```
